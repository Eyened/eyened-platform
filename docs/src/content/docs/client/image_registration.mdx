---
title: Image registration
description: Enable image registration in the viewer
---

The Eyened client can be configured to enable image registration for different image types, such as autoflourescence, infrared and colour fundus images.

## Setup

1. Add a new entry in the `FormSchema` table with `SchemaName` 'Pointset registration'

## Registrating images

1. After the registration feature is set up, a 'Registration' panel will be added in the right sidebar to the image viewer.
1. Click on the 'Create new registration set' button to start editing a new registration set.
1. At least 3 points need to be selected on each image.
